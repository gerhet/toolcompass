Was ist ein QR-Code und wie funktioniert er?

QR steht für "Quick Response" – und genau das ist der Kern dieser Technologie. Ein QR-Code ist ein zweidimensionaler Barcode, der Informationen in einem Raster aus schwarzen und weißen Punkten speichert. Anders als herkömmliche Strichcodes kann ein QR-Code nicht nur Zahlen, sondern auch Text, URLs, Kontaktdaten, WLAN-Zugangsdaten und mehr kodieren.

Die Daten werden mit Fehlerkorrektur gespeichert. Das bedeutet: Selbst wenn ein Teil des Codes beschädigt oder verdeckt ist, lassen sich die Informationen oft noch lesen. Je höher die Fehlerkorrekturstufe, desto mehr des Codes kann beschädigt werden – allerdings wird der Code dann auch größer.

Technik-Fakt: Ein QR-Code kann bis zu 7.089 Ziffern, 4.296 alphanumerische Zeichen oder 2.953 Bytes Binärdaten speichern – mehr als genug für die meisten Anwendungsfälle.

Die 7 beliebtesten Anwendungsfälle für QR-Codes

  1. Website-Links: Direkter Zugriff auf Landingpages, Produkte oder Social-Media-Profile
  2. Kontaktdaten (vCard): Visitenkarten digitalisieren – scannen und Kontakt speichern
  3. WLAN-Zugang: Gäste scannen den Code und verbinden sich automatisch mit dem Netzwerk
  4. E-Mail & SMS: Vorausgefüllte Nachrichten zum schnellen Versenden
  5. Veranstaltungen (iCal): Termine direkt in den Kalender übertragen
  6. Zahlungen: PayPal- oder Crypto-Adressen kodieren
  7. Produktinfos: Zusätzliche Informationen zu physischen Produkten bereitstellen

Datenschutz beim QR-Code erstellen: Worauf achten?

Nicht jeder QR-Generator ist gleich. Viele kostenlose Online-Tools speichern Ihre Eingaben auf Servern oder leiten die gescannten Codes über Tracking-Dienste. Das birgt Risiken:

  • Ihre URLs oder Texte können in Datenbanken gespeichert werden
  • Tracking-Parameter werden manchmal automatisch hinzugefügt
  • Werbeanalysen können mitgelesen werden
  • Geschäftliche URLs oder interne Links werden potenziell preisgegeben

Die Lösung: Ein Generator, der 100% client-seitig arbeitet. Das bedeutet: Die QR-Code-Erzeugung findet ausschließlich in Ihrem Browser statt. Keine Daten werden an Server gesendet, nichts wird gespeichert.

Tipps für optimale QR-Codes

  • Kontrast ist wichtig: Schwarz auf Weiß funktioniert am besten. Vermeiden Sie helle Farben.
  • Mindestgröße beachten: Zum Drucken mindestens 2×2 cm, besser 3×3 cm.
  • Rand freilassen: Lassen Sie rund um den QR-Code einen weißen Rand von mindestens 4 Modulen.
  • Testen Sie vor dem Druck: Scannen Sie den Code mit verschiedenen Smartphones.
  • Kurze URLs bevorzugen: Je kürzer der Text, desto weniger dicht ist der Code und desto besser lässt er sich scannen.

FAQ: Häufige Fragen zu QR-Codes

Können QR-Codes Viren enthalten?

Der QR-Code selbst ist nur eine Textinformation. Gefährlich wird es erst, wenn die kodierte URL auf eine bösartige Website führt. Scannen Sie daher nur Codes aus vertrauenswürdigen Quellen – genauso, wie Sie keine Links aus unbekannten E-Mails anklicken würden.

Wie lange sind QR-Codes gültig?

Statische QR-Codes (wie unser Generator sie erstellt) sind zeitlich unbegrenzt gültig. Solange die kodierte URL oder der Text existiert, funktioniert der Code. Dynamische QR-Codes (mit Weiterleitungsservice) können hingegen ablaufen oder kostenpflichtig werden.

Kann ich einen QR-Code mit Logo erstellen?

Ja, das ist möglich – allerdings nur bei höherer Fehlerkorrektur (ab 15% / Level M). Das Logo wird in die Mitte platziert und der Code kompensiert den Verlust durch zusätzliche Redundanz. Diese Funktion planen wir für eine zukünftige Premium-Version.

Welche Fehlerkorrektur sollte ich wählen?

Niedrig (L/7%): Für saubere Drucke in kontrollierter Umgebung.
Mittel (M/15%): Der Standard für die meisten Anwendungen.
Hoch (Q/25%) oder Sehr hoch (H/30%): Für Codes, die stark beansprucht werden (Außenbereich, rauhe Oberflächen, Produkte mit Logo).
